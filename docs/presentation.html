01 — סקירה
מה זה DevTeam Hub?
פלטפורמת ניהול פיתוח משולבת המאחדת את כל הכלים הנחוצים לצוות פיתוח מקצועי

DevTeam Hub היא מערכת ניהול פיתוח תוך-ארגונית מלאה, המספקת לצוות הפיתוח לוח בקרה אחד לניהול מחזור החיים המלא של מוצר תוכנה — מהרעיון ועד הפריסה לסביבת הייצור.

המערכת פועלת על שרתים פנימיים בלבד, ומתממשקת עם כלים On-Premises כגון Gitea, Jenkins, SonarQube, Ollama, Grafana Loki ו-Jitsi — ללא שליחת נתונים רגישים לענן.

בנוסף, המערכת כוללת פורטל לקוחות נפרד, מערכת דוחות מלאה, CLI ו-VS Code Extension המאפשרים ניהול מלא מתוך ה-IDE ושורת הפקודה.

03 — כלים
כלים ותשתית On-Premises
כל הכלים פועלים על שרתים פנימיים ברשת הארגון — ללא תלות בשירותי ענן
🦊
Gitea — שרת Git
חלופה תוך-ארגונית ל-GitHub. ניהול repos, branches, PRs — מסונכרן ל-DevTeam Hub דרך Webhooks.
🔧
Jenkins — CI/CD
שרת אוטומציה לבניה, בדיקות ופריסה. כל build מדווח ל-DevTeam Hub — כולל סטטוס, תוצאות בדיקות ולוגים.
📹
Jitsi — וידאו
שרת וידאו-ועידה בקוד פתוח. פגישות צוות, Stand-ups — עם Join אוטומטי מתוך DevTeam Hub וייצוא iCal.
🔍
SonarQube — סקירת קוד
ניתוח סטטי אוטומטי עם Quality Gates, OWASP SAST, code smells. תמיכה ב-8+ שפות.
🤖
Ollama — AI Code Review
סקירת קוד מבוססת AI — 100% מקומית. מודל Qwen2.5-Coder 32B. סקירת diff לכל PR.
📋
Grafana Loki — לוגים
איסוף לוגים מרכזי בפורמט JSON אחיד. correlation IDs, סינון labels, 30 ימי retention, 256 MB RAM.
📊
Grafana — Dashboards
דשבורדים והתראות ללוגים. סינון errors ו-warnings בזמן אמת, LogQL queries, התראות.
🐛
Sentry — Error Tracking
מעקב שגיאות ו-exceptions בזמן אמת. Stack traces, breadcrumbs, performance monitoring.
🔒 יתרון ה-On-Premises:

כל הנתונים הארגוניים — קוד, כרטיסים, CI logs, deployment history, תקשורת פנימית — נשארים בתוך הרשת הארגונית. אין תלות ב-SLA של ספק ענן חיצוני, אין עלויות per-seat, ואין חשיפת קניין רוחני לצדדים שלישיים.

14 — אבטחה
אבטחת המערכת
8 שכבות אבטחה המגנות על הנתונים, הגישה והתקשורת בכל רמות המערכת
🔑
אימות (Authentication) — Devise
שני מודלי Devise נפרדים: User (צוות) + CustomerUser (פורטל). סיסמאות bcrypt. Password reset.
95%
🛡️
הרשאות (Authorization) — Pundit
Pundit policies על כל resource. היררכיית תפקידים: developer < team_lead < project_manager < admin / qa.
88%
🔒
CSRF Protection — Rails Built-in
כל form ו-AJAX מאומת עם CSRF token. Turbo שולח X-CSRF-Token אוטומטית.
100%
🪝
Webhook Validation — HMAC Header
כל Webhook מ-Gitea ו-Jenkins מאומת עם secret header. בקשות ללא חתימה נדחות (403).
90%
🏰
בידוד פורטל הלקוח
CustomerPortal::BaseController יורש מ-ActionController::Base — אין נגישות למשתנים פנימיים.
95%
🧹
ניקוי קלט — Strong Parameters
כל controller מגדיר permit מפורש. HTML בצ'אט עובר html_escape.
85%
🕐
ניהול Session — Devise
remember_me token מוצפן, timeout session, Rails Encrypted Cookies.
90%
🌐
Modern Browser Enforcement
allow_browser versions: :modern דוחה דפדפנים ישנים. CSP meta tags מגנים מפני XSS.
80%
